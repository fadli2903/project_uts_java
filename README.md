# Project UTS - Sistem Pengelolaan Kos & Kontrakan (Java)

**Penjelasan singkat**  
Aplikasi console Java untuk mengelola data Properti (KamarKos & Kontrakan) dan Penyewa. Menerapkan konsep OOP: encapsulation, inheritance, constructor parameter, association, dan polymorphism.

**Fitur**
- Menambah properti (KamarKos / Kontrakan)
- Menampilkan semua properti
- Mencari properti berdasarkan ID
- Mengubah harga properti
- Menambah penyewa (meng-assign KamarKos)
- Menampilkan penyewa
- Mengecek status properti (tersedia / tidak)
- Menghapus penyewa (checkout)
